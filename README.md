# BankSight

BankSight is a Streamlit application for reviewing and organizing bank transaction CSV files in a clean, browser-based dashboard.

It is designed for bookkeepers who manage multiple clients and need a fast way to upload statements, categorize transactions, and review summaries.

## Run locally

1. Create and activate a virtual environment.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Start the app:
   ```bash
   streamlit run app.py
   ```

## Streamlit Community Cloud

This repository is ready for deployment on Streamlit Community Cloud (`share.streamlit.io`) with:

- **Main file path:** `app.py`
- **Dependencies:** `requirements.txt`
